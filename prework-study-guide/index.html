HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML attributes are always defined in the beginning element, never the closing element
  • !DOCTYPE is not case sensitive because it is a declaration. Declarations are not case sensitive. HTML elements are case sensitive.
  • The terms tag represents the part of an element that is within the bracket, while element refers to the entire "element".

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS is a tool that makes websites more visually appealing
  • A popular way to use CSS is to use an external style sheet and then link it to an HTML document.
  • Class attributes alow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: initialize an existing directory as a Git repository.
  • git clone: retrieve an entire repository from a hosted location via URL
  • git add [file]: add a file as it looks not to your next commit stage.
  • git reset [file]: unstage a file while retaining the changes in a working directory.

JavaScript

  • JS is the tool that takes a staic HTML/CSS webpage and makes it interactive.
  • Variables are containers that sotore values.
  • You declare a variable with the let keyword, followed by the name you give the variable. ex. let myVariable;
  • A semicolon at the end indicates where the statement ends and is only required when you need to separate statements on a single line.
  • You can name a variable nearly anything but there are certain restrictions.
  • JavaScript is case sensitive meaning myvariable is not the same as myVariable.
  • After declaring a variable you can give it a value myVariable = "bob";
  • you can do these operations on the same line: let myVariable = "Bob";
  • you write a comment like so /* Everything in between the two starred forward slashes is a comment */
